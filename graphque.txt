Here are Neo4j Cypher queries to view the created graph and its relationships:

View all nodes and relationships (full graph):
MATCH (n)-[r]->(m) RETURN n, r, m

View all nodes (with properties):
MATCH (n) RETURN n

View all relationships (with properties):
MATCH ()-[r]->() RETURN r

View graph for a specific Service:
MATCH (s:Service {name: $serviceName})-[r*1..4]-(n) RETURN s, r, n

View node counts by type:
MATCH (n) RETURN labels(n)[0] AS nodeType, count(*) AS count

View all edges of a specific type (e.g., CALLS):
MATCH (a)-[r:CALLS]->(b) RETURN a, r, b

Finds all File nodes that define something (e.g., a Class, Function, etc.):
MATCH (f:File)-[:DEFINES]->(N) RETURN f, N

Finds all Service nodes that expose an Endpoint:
MATCH (s:Service)-[:EXPOSES]->(e:Endpoint) RETURN s, e

Files importing modules:
MATCH (f:File)-[:IMPORTS]->(m:Module) RETURN f, m

Functions calling other functions
MATCH (a:Function)-[:CALLS]->(b:Function) RETURN a, b

Services depending on other services:
MATCH (a:Service)-[:DEPENDS_ON]->(b:Service) RETURN a, b
